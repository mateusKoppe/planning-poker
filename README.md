# Planning Poker

## Setup
### Installing dependencies

On client:

```bash
cd client
yarn
```

On server:

```bash
cd server
yarn
```

### Developing
On client:

```bash
cd client
yarn start
```

On server:

```bash
cd server
yarn dev
```

### Building
On client:

```bash
cd client
yarn build
```

On server:

```bash
cd server
yarn build
```
